# Phantom Block Helper

Fabric-мод, автоматизирующий трюк "фантомный блок" для манхантов на Minecraft 1.21.11.

## Что делает
- По нажатию клавиши `'` (апостроф, можно поменять в настройках управления
  Minecraft — категория "Phantom Block Helper") включает/выключает режим.
- Пока режим включён: любой блок, который ты ставишь правым кликом,
  автоматически проходит последовательность F -> постановка из оффхенда ->
  F -> F, вместо обычной постановки.
- Ещё раз нажал клавишу — режим выключен, блоки ставятся как обычно.

## Сборка
Нужны: JDK 21, интернет-доступ (Gradle качает зависимости сам).

1. Открой папку проекта в терминале.
2. Проверь `gradle.properties` — версия `fabric_version` там placeholder,
   зайди на https://modrinth.com/mod/fabric-api?version=1.21.11 и подставь
   актуальную версию Fabric API под 1.21.11.
3. Собери:
   - Linux/macOS: `./gradlew build`
   - Windows: `gradlew.bat build`
4. Готовый jar будет в `build/libs/phantomblock-1.0.0.jar`.
5. Установи Fabric Loader + Fabric API нужной версии на клиент и закинь
   jar в папку `mods`.

Проще всего собирать через IntelliJ IDEA с плагином Minecraft Development
или просто открыв проект как Gradle-проект — тогда все шаги делаются кнопками.

## Способ 2: собрать в облаке через GitHub (если у тебя проблемы с интернетом/VPN)

Не нужен ни Java, ни Gradle на компьютере — сборка идёт на серверах GitHub.

1. Зайди на https://github.com и зарегистрируйся (бесплатно, email + пароль).
2. Нажми "+" в правом верхнем углу → "New repository". Назови как угодно,
   например `phantomblock-mod`, поставь "Public", нажми "Create repository".
3. На странице пустого репозитория нажми "uploading an existing file".
4. Перетащи туда **всё содержимое** папки `phantomblock-mod` с компьютера
   (все файлы и папки: build.gradle, gradle.properties, settings.gradle,
   README.md, папку src, папку .github).
5. Внизу нажми "Commit changes".
6. Открой вкладку "Actions" вверху репозитория — там автоматически
   запустится сборка (значок жёлтого/зелёного кружка). Подожди 2-5 минут,
   пока не станет зелёная галочка.
7. Кликни на завершённый запуск → внизу страницы будет раздел "Artifacts" →
   там файл `phantomblock-mod` — скачай его (это zip с готовым jar внутри).
8. Распакуй, внутри будет `phantomblock-*.jar` — это и есть мод, дальше
   как в шаге 5-6 выше (закинуть в папку `mods`).


Открой `PhantomBlockMod.java`, в начале файла есть константа:

```java
private static final int TICKS_BETWEEN_STEPS = 0;
```

Поставь 1 или 2 — это добавит паузу между шагами (свап/постановка/свап/свап),
что может быть нужно на серверах с большим пингом или тик-лагом.

## Важно
- Это чисто клиентская фича, использующая штатные пакеты игры
  (свап руки, постановка блока) — просто в нужной последовательности
  и без ручного тайминга. Работает только в паре с Fabric API на клиенте.
- Если сервер как-то валидирует/античит эту механику — трюк может не
  сработать вообще, это не баг мода, а особенность сервера.
